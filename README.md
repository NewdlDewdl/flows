# Flows

Three interval-timer HTML flows for movement work — built for Rohin, hosted via GitHub Pages for stable Safari rendering.

Live: **https://newdldewdl.github.io/flows/**

- **Hip Awakening** — ~48 min, late-night, parasympathetic ending (sleep prep). Hip-focused.
- **The Centering Primer** — ~35 min, daytime, sympathetic ending (cold splash, walk into the day). Hip bloom + HIIT cascade.
- **The Upper Recovery** — ~20 min, anytime, parasympathetic ending. Cervical + thoracic + scapular focus with light hip reset.

Each timer runs offline once loaded (audio chimes via Web Audio API, wake-lock to prevent screen sleep, dark warm theme).

Source files are in `~/fitness-program/` on Rohin's Mac. Routing rules and design rationale live in `~/.claude/projects/-/memory/feedback_contortion_session_setup.md`.
